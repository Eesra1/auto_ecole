# auto_ecole
Ce projet permet de gérer les candidats, les moniteurs, les véhicules, les séances de code et de conduite ainsi que le suivi comptable d’une auto-école. L’application inclut également la planification des séances, la gestion des maintenances des véhicules et le suivi des paiements des candidats.


##  Description

Cette application permet de gérer les différentes activités d’une auto-école :

- Gestion des candidats
- Gestion des véhicules
- Gestion des moniteurs
- Gestion des séances de code et de conduite
- Gestion comptable
- Planification des séances
- Suivi des maintenances des véhicules

Le projet est basé sur les principes de la programmation orientée objet afin d’assurer une architecture claire, modulaire et maintenable.

---

## Fonctionnalités

###  Gestion des candidats
- Ajouter un candidat
- Modifier les informations
- Supprimer un candidat
- Rechercher un candidat
- Consulter :
  - Prix total
  - Montant payé
  - Reste à payer

###  Gestion des véhicules
- Ajouter / modifier / supprimer un véhicule
- Gestion des types :
  - Moto
  - Voiture
  - Camion
  - Autobus
- Suivi :
  - Assurance
  - Visite technique
  - Vignette
  - Vidange
- Historique des réparations et maintenances

###  Gestion des moniteurs
- Gestion des moniteurs
- Calcul du salaire mensuel
- Gestion des disponibilités
- Planification des séances

###  Gestion des séances
- Séances de code
- Séances de conduite
- Gestion des rendez-vous
- Affichage du planning hebdomadaire

###  Suivi comptable
- Gestion des dépenses
- Gestion des revenus
- Suivi mensuel



##  Technologies utilisées

- Java
- Programmation Orientée Objet (POO)
- JSON pour la sauvegarde des données
